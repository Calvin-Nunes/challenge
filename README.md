# FUTURE - challenge

# Opção 1: via CMD

```bash
# INSTALL
$ npm install

# RUN APP
$ npm run dev
```

# Opção 2: via .bat
```
Dentro da pasta raíz, há 2 arquivos .bat
1. install.bat
2. run_app.bat

Executar eles na ordem acima
```

Após utilizar uma das opções, abrir o browser no endereço: localhost:3000
